# iOS-WeatherApp
Simple location based weather app
 - Provides the current temperature
 - Shows the five day forecast
 - Location is hard coded to Irvine, CA (will most likely change this in the future)
 
Swift Practice
 - AutoLayout (Stack Views)
 - REST APIs
 - Alamofire
