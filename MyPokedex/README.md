# iOS-MyPokedex
Describes each of the 718 pokemon in the pokemon universe
 - Has search filter to help search through the pokemon
 - Using the data from the online [Pokemon API](http://pokeapi.co/)
 
Swift Practice
 - Auto Layout
   - Stack Views
 - Collection View
 - Alamofire
